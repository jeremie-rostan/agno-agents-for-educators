# School Leader Apps

AI-powered applications designed specifically for school administrators, principals, and educational leaders. These tools help streamline administrative tasks, track student support needs, and make data-driven decisions to improve school operations.

## Available Applications

### [Student Support Ticket System](https://github.com/jeremie-rostan/Student-Support-Ticket-System)

A comprehensive Next.js application for tracking and managing student support tickets with AI-powered analysis, voice transcription, and pattern recognition.

**Key Features:**
- Create and manage student support tickets with multiple students per ticket
- Categorize tickets (Academic, Behavioral, SEL, or custom categories)
- Track ticket status (New, In Progress, Resolved)
- Add conversation notes with voice recording and transcription
- AI Assistant powered by LM Studio for analyzing patterns and generating reports
- Student profiles showing all tickets and patterns for individual students
- Voice recording with Web Speech API and AssemblyAI integration
- Dark mode support and mobile-friendly design

**Tech Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS, shadcn/ui

**Perfect for:**
- Principals and assistant principals tracking student interventions
- Counselors managing student support cases
- Dean of students documenting behavioral incidents
- School leadership teams analyzing student support trends

## Why AI for School Leadership?

School leaders juggle countless responsibilities. AI-powered tools can help by:
- Automating routine administrative tasks
- Identifying patterns in student behavior and support needs
- Generating comprehensive reports for stakeholders
- Streamlining communication and documentation
- Making data-driven decisions to support students and staff

These applications are designed to protect student privacy, run locally when possible, and integrate seamlessly into existing school workflows.
